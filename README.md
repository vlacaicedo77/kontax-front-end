# Sifae 2.0 vista

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.24.

## Clonar proyecto

Ejecutar el siguiente comando

- git clone https://github.com/TrazabilidadPROAmazonia/sifae_vista.git sifae

## Instalaciones necesarias

- Navegador web.
- NodeJS versión 8 o superior [NodeJS](https://nodejs.org/es/)
- Visual Studio Code 
- AngularCLI versión 10 o superior [AngularCLI](https://cli.angular.io/)

## Proceso de ejecución

- Instale los programas requeridos en el paso anterior.
- Descargar el presente proyecto y ubicarlo en un directorio que disponga de permisos completos para inicializarlo.
- Mediante el terminal acceda al directorio del proyecto.
- Ejecute el siguiente comando para inicializarlo: npm install 
- En la misma ubicación del proyecto ejecute el siguiente comando para ejecutar el proyecto: ng serve -o

## Inicializar proyecto con GIT

Acceda al directorio del proyecto y ejecute los siguientes comandos:
